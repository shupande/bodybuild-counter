# bodybuild-counter

受万恶的美帝电影影响，男女都对大胸部深有喜爱，经过老衲一番Google选出了练就大胸Ji和减肚子的三大方法。

由于练习时心里默数会导致分神，遂产生本想法，算是学习VueJs和Cordova的练手。



> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
